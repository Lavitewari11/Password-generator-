# 🔐 Password Generator

A simple and responsive Password Generator built using HTML, CSS, and JavaScript. This project helps users generate strong and secure passwords with customizable options to improve online security.

## 🚀 Features

* Generate secure random passwords
* Adjustable password length
* Include uppercase letters (A-Z)
* Include lowercase letters (a-z)
* Include numbers (0-9)
* Include special characters (!@#$%^&*)
* Copy password to clipboard
* Responsive user interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

```bash

```

### Run the Project

Open `index.html` in your browser.
